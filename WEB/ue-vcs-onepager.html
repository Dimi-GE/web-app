Component Usage Guide

// Container Classes

.container - Standard container (max-width: 1200px)

.container-narrow - Narrow container (max-width: 800px) - for focused content

.container-wide - Wide container (max-width: 1400px)

// Section Classes

.section - Standard section padding (6rem vertical)

.section-sm - Small section padding (3rem vertical)

.section-lg - Large section padding (6rem vertical)

Combine with background classes:

.bg-dark - Dark background (#0A1628)

.bg-elevated - Slightly lighter background

.bg-card - Card background color

// Grid System

.grid - Base grid container

.grid-2 - 2 column responsive grid (min 300px per column)

.grid-3 - 3 column responsive grid (min 250px per column)

.grid-4 - 4 column responsive grid (min 200px per column)

All grids automatically collapse to single column on mobile

// Flexbox Utilities

.flex - Display flex

.flex-center - Center items (justify + align)

.flex-between - Space between items

.flex-col - Flex direction column

.gap-sm / .gap-md / .gap-lg - Add gaps between flex/grid items

// Typography Classes

.heading-hero - Main hero heading with gradient

.heading-section - Section titles

.heading-subsection - Subsection titles (cyan color)

.heading-card - Card titles

.text-large - Large body text (1.25rem)

.text-base - Standard body text (1rem)

.text-small - Small text (0.875rem)

.text-accent - Orange accent color with bold weight

Alignment classes:

.text-center / .text-left / .text-right - Text alignment

// Button Classes

.btn - Base button class (required)

.btn-primary - Cyan gradient button

.btn-secondary - Cyan outline button

.btn-accent - Orange gradient button

.btn-ghost - Minimal transparent button

Size modifiers:

.btn-sm - Small button

.btn-lg - Large button

// Card Classes

.card - Standard card with hover effects

.card-feature - Feature card with gradient and top border accent

// Special Components

.icon-box - Icon container with gradient background

.badge - Base badge class

.badge-primary / .badge-accent / .badge-outline - Badge variants

.input - Form input with focus states

.input-group - Input + button combo

.cta-box - Call-to-action box with pulse animation

// Background Effects

.circuit-bg - Circuit board pattern overlay

.glow-orb - Glowing orb background effect

.glow-orb-cyan / .glow-orb-orange - Color variants

Position these absolutely within a relative parent

// CSS Variables Reference

Spacing: --space-xs, --space-sm, --space-md, --space-lg, --space-xl, --space-2xl, --space-3xl

Border Radius: --radius-sm, --radius-md, --radius-lg, --radius-xl

Transitions: --transition-fast, --transition-base, --transition-slow

Use these in your custom styles: padding: var(--space-lg);

// Example Usage

<section class="section bg-dark">
  <div class="container">
    <h2 class="heading-section text-center">Your Title</h2>
    <div class="grid grid-3">
      <div class="card">
        <h3 class="heading-card">Feature 1</h3>
        <p class="text-base">Description</p>
      </div>
      <!-- More cards -->
    </div>
  </div>
</section>

Version Control That Thinks Like You Do

Stop wrestling with Perforce pricing and Git commands. Your Unreal Engine projects deserve version control that just... works. Automatically.

Join Beta Testing See How It Works

The Problem Every Solo Dev Knows

You're building something amazing in Unreal Engine, but backing up your work shouldn't require a computer science degree or an enterprise budget. Pay thousands for Perforce, or battle Git's learning curve while manually tracking every single change.

Meanwhile, you just want to focus on creating. Not configuring version control systems.

How It Works

Three steps. Zero learning curve. Complete protection.

1️⃣

Install Plugin

Download from FAB Marketplace and activate. The plugin creates its workspace and initializes your project automatically.

2️⃣

Work Normally

Create, edit, delete files like you always do. No manual commits. No terminal commands. The tool tracks everything in real-time.

3️⃣

Stay Protected

Automatic backups at your chosen intervals. When something breaks, restore it in seconds. That's it.

Why This Changes Everything

🚫

No Terminal Required

Forget git commands. Forget manual commits. The tool speaks your language: you make a change, it's tracked. Instantly.

⚡

Real-Time Tracking

Event-driven system subscribes to Unreal Engine's file operations. Every create, modify, or delete is captured automatically.

⏱️

Smart Backup Timing

Configure time-based backups (Pomodoro style) or let it work automatically. Optional restore functionality coming in v1.0.
